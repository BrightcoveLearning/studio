Exporting Data from the Analytics Module

This topic describes how you can export report data from the Analytics module.

For an introduction to the Analytics module, see Getting Started with the Analytics Module.

Exporting report data

To export data from a report, follow these steps.

  1. Navigate to the Analytics module.
  2. Use the date selector to select a time span.
  3. Select a report.
  4. Click the appropriate link to export the data as a CSV or XLSX file.
  5. Save the data file.
