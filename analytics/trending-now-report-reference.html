Trending Now Report Reference

In this topic you will learn about the Trending Now report that is provided as part of the Analytics module.
Note: This feature is only enabled for previous Ooyala customers who have migrated to a Video Cloud account.

Analytics data is sent automatically by Brightcove Players, including those provided by the Native Player SDKs. If you are not using a Brightcove Player to deliver Video Cloud videos, you need to instrument the player you are using to send the data to the Data Collector. See the Overview: Data Collection API topic for more details.

To view the Trending Now report, open the Analytics module and select Trending Now from the left side navigation. The report will display a graph showing the number of concurrent viewers for the past hour. The graph will update every 30 seconds.

Note: A unique user is determined by browser user agent combined with IP address. Multiple tabs of the same player running in the same browser will only be 1 user.

If your account is enabled for Brightcove Live, you will have the option to view the number of concurrent viewers for all videos, VOD assets or live assets.

Sections below the graph will display the Top Videos, Top Countries and Top Cities for the last minute and will update every 10 seconds.
