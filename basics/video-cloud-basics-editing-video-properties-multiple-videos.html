Video Cloud Basics: Editing Video Properties for Multiple Videos

In this topic you will learn how to use the Quick Edit feature in the Media module to edit video properties for multiple videos.
 

Steps

  1. Open the Media module.
  2. Select the videos to edit by clicking the checkbox next to the video name.
  3. Click Quick Edit.
  4. Use the Quick Edit panel to edit the video properties as needed.
  5. Click Save to save the changes.
