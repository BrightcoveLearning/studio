Using Cloud Playout to Stream VOD Videos

In this topic you will learn how to create a collection of VOD videos that can be streamed as a linear channel.

The Cloud Playout module enables broadcasters and media publishers to create a collection of VOD videos and Live events and then stream it as a linear channel and leverage all the features provided by Brightcove Live.

Note: This feature is currently in Limited Availability. Contact your Brightcove Account Manager for more information.

Creating a new channel

To create a new Cloud Playout channel, follow these steps.

  1. Login to Video Cloud Studio.
  2. Click Cloud Playout in the navigation header. A list of all Cloud Playout channels that have been created will be displayed (if any).
  3. Click Create New.
  4. Enter the channel Identification details:
    • Name - Name for the channel
    • Description - Channel description
    • Image URL - URL to an image file for the channel; meant to be a channel image that could be retrieved as part of the channel's metadata
  5. Enter the channel Schedule details.
    • Loop Playout Content - If On,
    • Enable Server Side Ad Insertion(SSAI) -
    • Start Date/Time -
    • End Date/Time -
  6. Enter the channel Destination details.
    • Mobile & Web - If checked, will optimize the display for mobile and web display
    • Smart TV - If checked, will optimize the display for Smart TV display
    • Output Region -
    • RTMP - Select to output the stream to an RTMP URL and then enter the RTMP URL to use
    • Brightcove Live Event - Select to output the stream to a Brightcove live event and then select a live event.
    • S3 Bucket - Select to output to an Amazon S3 bucket. Enter the bucket details
